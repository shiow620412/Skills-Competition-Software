Problem 1296. -- 107計算機程式設計作業-09-Selection sort

1296: 107計算機程式設計作業-09-Selection sort

Time Limit: 1 Sec  Memory Limit: 128 MB
Submit: 156  Solved: 59
[Submit][Status][Web Board]

Description

Given a series of integers, please write a selection sort function to sort the input integers and output in an non-increasing order.

Input

先輸入將被排序的整數個數K，然後，再輸入K個整數。

Output

將輸入的K個整數做Selection sort，然後以降冪方式依序輸出K個整數。

Sample Input

10
2 3 5 6 8 9 7 4 1 0
6
-5 -3 0 2 1 7
9
0 0 -1 5 6 -7 -8 -0 10

Sample Output

9 8 7 6 5 4 3 2 1 0
7 2 1 0 -3 -5
10 6 5 0 0 0 -1 -7 -8
